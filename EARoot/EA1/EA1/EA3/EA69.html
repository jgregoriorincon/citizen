-- Definición --<br /></p><p>Cualquier número de unidades espaciales, considerado como una entidad.<br /></p><p><br /></p><p>-- Descripción --<br /></p><p>Para la Base de Datos de Ciudadanos de Bogotá, se contempla esta clase como el conjunto de unidades espaciales. Dado que la unidad espacial es el "lote catastral";  cualquier conjunto de lotes como manzana, sector catastral, municipio, etc, es considerado aquí.<br /></p><p><br /></p><p>-- Fuente --<br /></p><p>Adaptado de: International Standard ISO/FDIS 19152 Geographic information - Land Administration Domain Model (LADM)<br /></p>
